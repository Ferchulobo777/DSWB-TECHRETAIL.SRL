## 📌 Descripción del Proyecto

Sistema backend desarrollado con Node.js, Express y MongoDB para la gestión de una plataforma e-commerce autogestionada. Permite administrar usuarios, productos y pedidos, aplicando relaciones entre entidades y validaciones de negocio.

---

## ⚙️ Funcionalidades

- CRUD de usuarios  
- CRUD de productos  
- CRUD de pedidos  
- Relación entre entidades mediante ObjectId  
- Validación de stock  
- Cálculo automático de total  
- Uso de populate para consultas relacionales  

---

## 🛠️ Tecnologías utilizadas

- Node.js  
- Express  
- MongoDB Atlas  
- Mongoose  

---

## 🔗 Endpoints principales

- POST /usuarios  
- POST /productos  
- POST /pedidos  
- GET /pedidos
